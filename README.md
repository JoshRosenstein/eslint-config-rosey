My ESLint Config 



## Installation

---


### Instal useing:

``` bash
# For yarn:
yarn add -D eslint eslint-config-rosey

# For npm:
npm i --save-dev eslint eslint-config-rosey
```


## Usage
---


``` json
{
  "extends": ["rosey"]
}
```

Or

``` json
{
  "extends": ["eslint-config-rosey"]
}
```
